AppCtrl

Description

App controller. Holds all $rootScope functions $rootScope.profileSettings = user's internet settings, name, waterwork. Stored in IDB 'settings'-table References: logout.png https://commons.wikimedia.org/wiki/File:Logout.svg

Called by: AddTaskCtrl.js ManageGroupCtrl.js StatisticsCtrl.js

HTMLpages used: AddTask.html JoinGroup.html Login.html ManageGroup.html NewUser.html Statistics.html Tabs.html
