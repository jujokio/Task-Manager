StatisticsCtrl

Description

Controls AddTask.html Calls AppCtrl.js if needed

Display own goup's statistics. Stats are divided in categories and displayed for every group member. Display every other groups total work hours

Methods

  • createOtherChart()

    do charts for other groups

  • createOwnChart(memberlist, series)

    do charts for own group X-axis: Member names, Y-axis: workhours Member names are in same order as workhours "a" did X hours in "Coding" Is deferred no reject

    Parameters
    ParamTypeDetails
    memberlistArray

    list of groupmember names ["a","b",...]

    seriesArray

    list of workhours [{"name":"Coding", "data":X},{"name":"coding", "data":Y},... ]

  • getMemberName()

    Display group member's name Is deferred, no rejects

    Returns
    Array

    array of names

  • init()

    init, fired every time when browsed to the Statistics tab Calls askAPI()
